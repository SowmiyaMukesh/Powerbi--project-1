## End-to-End Business Performance Dashboard (Power BI)

# Project Overview

In this project, I worked on a large business dataset and built an end-to-end Power BI dashboard to analyze performance across multiple business functions including Finance, Sales, Marketing, and Supply Chain.
Since the dataset was large, I first handled the data in SQL, structured it, and then connected it to Power BI. From there, I performed data transformation, cleaning, and modeling before building an interactive dashboard.
The final report provides a complete view of business performance with detailed insights into sales, profitability, forecast accuracy, and overall trends.
________________________________________
# Data Source & Preparation


The dataset was initially processed in SQL due to its size and complexity. After organizing the data, it was imported into Power BI for further processing.
In Power BI, I performed:
•	Data transformation using Power Query (ETL process)
•	Data cleaning and formatting
•	Creation of relationships between fact and dimension tables
•	Data modeling to support multi-dimensional analysis
The data model includes structured fact and dimension tables such as customer, product, market, and date.

# DAX Measures & Calculations

To support analysis and KPI tracking, I created multiple DAX measures across the report.
Key measures include:
•	Net Sales ($)
•	Gross Margin ($ and %)
•	Net Profit ($ and %)
•	Forecast Accuracy %
•	Net Error and ABS Error
•	Market Share %
•	Operational Expense ($)
•	P&L Metrics (values, change, and targets)
These measures were used to compare performance across different time periods, benchmarks, and targets.
________________________________________
# Dashboard Structure

The dashboard is organized into multiple views for better analysis:
•	Finance View → Profit & Loss analysis, cost breakdown, profitability
•	Sales View → Customer and product performance, sales trends
•	Marketing View → Product-level performance and profitability insights
•	Supply Chain View → Forecast accuracy, error analysis, and risk indicators
•	Executive View → High-level business overview across all functions
Each view is interactive and connected using slicers such as year, quarter, region, customer, and product category.
